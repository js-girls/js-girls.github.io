js-girls is a 1-day workshop for girls who want to learn Javascript, one of the most popular languages for web development.

In 24 hours you will learn the basics of programming and get an introduction to the world of today's Javascript technology. You will build and interactive web interface using Javascript, HTML and CSS.

No prior knowledge of programming languages is required. Just basics familiarity with a computer.

Our awesome coaches will guide you through each step and make you build the sample application.
